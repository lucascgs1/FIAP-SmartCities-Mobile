## dependencias
- [nodejs 32bits]
- [angular]
- [ionic]

```
para executar o projeto é necessario abrir a pasta do projeto no cmd
executar o comando "ionic serve"
```

uma pagina no navegador apontando para o localhost:8100 ira abrir com o app

### Apk
na pasta do projeto, à uma pasta com nome apk com uma apk do projeto
